  但你的 LLM 增强文本是高质量的均匀数据，问题只是太短（被 CLIP 77 token 限制了）。

  所以正确做法是：

  ▎ 重新用 LLM 生成更长的语义画像（200-300 词），搭配 BGE/E5，扔掉原始 description。

  LLM 文本可控、均匀、无噪声，比原始 description 好得多。而且你已经跑过一次 LLM 增强了，只是 prompt
  里的长度限制太小。改一下 prompt 重新生成即可。


  - Word2Vec 只看局部共现（窗口内物品），丢了全局图结构
